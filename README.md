## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Customization

- Update personal information in the component files
- Modify colors in `src/index.css` (CSS variables)
- Add your own projects in `src/components/Projects.jsx`
- Update social media links in `src/components/Hero.jsx`
- Customize contact information in `src/components/Contact.jsx`

## Sections

- **Hero**: Introduction with name and titles
- **About**: Personal information and approach
- **Skills**: Technical skills and technologies
- **Projects**: Portfolio of featured projects
- **Contact**: Contact form and information
